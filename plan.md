# Rosca

## To be implemented
In this project a **Simple Rosca** will be implemented. The folowing functionalities must exist:

1. Add new customers
2. Create groups that add customers
3. Set a minimum number of customers a group can have
4. Decide first borrower
5. Create mechanism to rotatively decide who is the next borrower.
6. Add payment frequency to the group
7. Automated schedules must pick decided frequencies and process payments.
8. Payments should be an external library as it may be processed in future by different methods/mechanisms/entities.

## Development tools

### Language
This project will be developed in Java with Spring boot.

### Database
MySQL database on a docker instance
